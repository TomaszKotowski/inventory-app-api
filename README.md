# inventory-app-api
===
Server for frontend Inventory app 

##Step 0: Install required packages
```angular2html
npm install
```

##Step 1: Generate DB

This command create new DB for our API
```angular2html
npm run create
```

##Step 2: Run API server

```angular2html
npm start
```

## Configuration
`config.json`
```angular2html
{
  "SECRET_TOKEN": "Alamakota",
  "SERVER_PORT": 3400,
  "AUTH_REQUIRED": true
}
```
